Campaign 4 epic and final completion gate for the Modern Replacement claim.

## Required evidence

- [ ] Portable Tier 1 codec profile is C/FFI-free by dependency-graph proof.
- [ ] Primary read/write and compatibility read-only format profiles pass their exact matrices.
- [ ] OCI and package conformance profiles pass.
- [ ] Stable Rust API, C ABI, CLI exit-code contract, and support matrix are frozen and checked.
- [ ] Three-producer/two-consumer interoperability evidence exists per format/method.
- [ ] Malformed, fuzz, resource-arithmetic, symlink-race, and decompression-bomb gates pass.
- [ ] 10 GiB streaming soak stays within the documented RSS budget.
- [ ] Native and portable performance gates pass without unapproved sustained regressions.
- [ ] Portable/native, no_std, WASI inspection, big-endian, 32-bit, MSRV, and all-features CI pass.
- [ ] At least two release candidates complete all technical gates.

## Decision rule

Only after every checkbox has durable evidence may the project describe a 1.0
as a Modern Replacement. The issue does not authorize release execution; that
remains a separate, explicitly manual maintainer decision.
