Campaign 2 epic for validating mainstream package containers as composed archive
profiles.

## Outcomes

- Debian and RPM validators cover their real gzip, xz, zstd, and bzip2 payload
  combinations and container invariants.
- APK, JAR, IPA, MSIX, NuGet, Wheel, and EPUB validators report structural,
  method, signature/encryption, and metadata support precisely.
- Validators consume bounded events and do not require extracting untrusted
  packages.

## Acceptance

- [ ] Each profile has normative-spec references and independent producer fixtures.
- [ ] Unsupported methods produce structured capability errors.
- [ ] Malformed nesting, duplicate paths, traversal, and decompression bombs are tested.
- [ ] Support matrix distinguishes “container readable” from “profile valid”.
- [ ] Profile APIs and CLI JSON share typed findings and stable severity semantics.
