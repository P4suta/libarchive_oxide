## Tracking policy

Each independently mergeable outcome gets a child Issue created from the
Modern Replacement Issue Form and linked to its epic. An epic closes only when
its acceptance evidence is durable. Calendar dates and version numbers cannot
override the RM-400 gates.

This tracker does not authorize tags, publishing, release workflow dispatches,
GitHub Release edits, or asset uploads.
